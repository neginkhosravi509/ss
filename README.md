BookVision is a Windows Forms application designed to manage a school library through a clean and user-friendly interface.  
The program provides separate sections for book searching, member registration, and book management, along with a secure manager login panel.

Features

- Book Search: Search by title, author name, subject, or publication year.  
- Member Management: Register and manage library members.  
- Book Management: Add new books and update existing records.  
- Manager Login: Secure access for administrators to manage the system.  

BookVision offers a simple and efficient way to organize and operate a school library.
